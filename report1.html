Idea

Find Me is a web application built for all wi-fi devices that facilitates meet-ups. Not only does it give users meeting place suggestions, it also shows real-time trajectory maps, calculates ETAs, gives access to a built-in chat, and incorporates fun geolocation games for users that are on-the-go. Find Me can also be used on the desktop to ensure that your loved ones get to where they need to go safely and with that grocery list on hand! Find Me is perfect for your next impromptu meeting.

UI & Usability

  • Fixed navigation bar stays at the top of the page for quicker navigation
  • Top navigation is clearly label with a readable font
  • Meet page has circle tap/click navigation for easier scrolling
  • Settings page uses accordion style buttons for mobile screens
  • Dropdown chat can be accessed in the Meet and Games page
  • Scales for mobile, tablet, and desktop
  • Color scheme is neutral with one accent color for aesthetic appeal
  • Facebook and Google log-in allow for quick registration

Responsiveness

  • Mobile: One column scrolling
  • Tablet: Two column with no margins
  • Desktop: Two column with margins

Product Features
  • Individual Maps: In browser geolocation allows the user to input their location for others in the meetup to see where everyone is coming from
  • ETA Estimates: Using GoogleMap's distance/time estimates with traffic allows users to plan their time accordingly
  • Real-time Trajectory:Push notifications update maps in "real-time" so that everyone can see where everyone else in the meetup is
  • Chat: Built in chat allows users to speak to each other without needing external applications
  • Suggestions:Meeting place suggestions allow users to discover new restaurants, parks, and locations in their city
  • Games: Real maps allow users to choose areas for their Scavenger Hunt and Hide & Seek games
  • Unique IDs: Users can create a meet-up without an account for fast access on-the-go
  • History: User accounts can be created to retain a history of past meet-ups, such as that cool coffee shop you visited last week
  • Future Meet-Ups: Allows you to schedule future meet ups with friends and save the scheduled meeting to iCal or Google calendar
  • Linked Accounts: Quick sign-in's with Facebook and Google+ allow users to integrate accounts and see friend's that are already using the app
  • Web Application: Desktop UI allows for stay at home parents to make sure their child met up with Uncle Ben successfully
  • Meeting Notes: With an account users can access meeting notes, a perfect place to place that grocery list for your forgetful husband
  • Customization: Privacy settings and profile changes can be made, as well as different color schemes for personal customization
